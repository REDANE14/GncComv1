  • 1.1 OS INSTALLATION WINDOWS 10

FIGURE 1

MAKE SURE YOUR COMPUTER MEETS THE MINIMUM HARDWARE SPECIFICATIONS LISTED BY MICROSOFT FOR THE WINDOWS VERSION YOU'RE INSTALLING BEFORE BEGINNING THE WINDOWS OS INSTALLATION.

STEPS 1

Insert the USB drive: Insert the bootable USB drive into your computer's USB port.

Boot from USB drive: Restart your computer and enter the BIOS setup. Change the boot order

Begin the Windows installation process: Once your computer has successfully booted from the USB drive, you should see the Windows 10 setup screen. Click the "Install Now" button to

Enter the product key: Enter the product key for Windows 10 when prompted. You can either enter the product key now or skip this step and enter it later.

Choose the installation type: Select the "Custom" installation type to perform a clean installation of Windows 10.

Select the hard drive to install Windows 10: Choose the hard drive where you want to install Windows 10. You can create new partitions or delete existing ones if needed.<< /h2>

Begin the installation process: Click the "Next" button to start the installation process. Windows 10 will now be installed on your computer.

Set up Windows 10: Once the installation process is complete, you will be prompted to set up Windows 10. Follow the on-screen instructions to set up your user account and personalize your settings.
